
## Getting Started
Clone the repo and run npm i 
Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
I have used Nextjs , TailwindCSS, MaterialUI and various animation from Framer motion and react-reveal
